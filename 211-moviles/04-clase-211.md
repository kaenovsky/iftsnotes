## Estructura switch

```java

switch(signo) {
	case "Aries":
		System.out.println("Aries, 21 de marzo - 19 de abril");
		break;
	case "Tauro":
		System.out.println("Tauro, 20 abril - 20 de mayo");
		break;
	case "Geminis":
		System.out.println("Gemini, 21 de mayo - 20 de junio");
		break;
	default:
		System.out.println("No es un signo válido");
}

```
